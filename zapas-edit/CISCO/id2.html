Konfiguracja VLAN


Zadanie Polecenia IOS
Przejdź do trybu konfiguracji globalnej. S1# configure terminal
Przejdź w tryb konfiguracji interfejsu dla SVI. S1(config)# interface vlan 99
Skonfiguruj adres IPv4 interfejsu zarządzania. S1(config-if)# ip address 172.17.99.11 255.255.255.0
Skonfiguruj adres IPv6 interfejsu VLAN. S1(config-if)# ipv6 address 2001:db8:acad:99::1/64
Skonfiguruj prefix IPv6 interfejsu VLAN. S1(config-if)# ipv6 address fe80::4 link-local
Włącz interfejs VLAN. S1(config-if)# no shutdown
Wróć do trybu uprzywilejowanego EXEC. S1(config-if)# end
Zapisz bieżącą konfigurację do konfiguracji startowej. S1# copy running-config startup-config

Przypisanie VLAN`u do portów.


Zadanie Polecenia IOS
Informacje o VLAN`ach i przypisanych do nich portów. S1# show vlan
Przejdź do trybu konfiguracji globalnej. S1# configure terminal
Rozpoczecie konfiguracji portów FastEthernet od 1 do 2. S1(config)# interface range Fa0/1-2
Konfiguracja trybu członkostwa. S1(config-if-range)# switchport mode access
Przypisanie VLAN`a 99 do portów z zakresu. S1(config-if-range)# switchport access vlan 99
Zakończ konfigurację S1(config-if-range)# exit
Konfiguracja pojedyczego portu. S1(config)# interface Gig0/1
Zakończ konfigurację S1(config-if)# exit
Konfiguracja trybu członkostwa. S1(config-if-range)# switchport mode access
Przypisanie VLAN`a 99 do portu. S1(config-if-range)# switchport access vlan 99
Zakończ konfigurację S1(config-if)# end
Informacje o VLAN`ach i przypisanych do nich portów. S1# show vlan
