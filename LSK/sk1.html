Elementy składowe sieci

W sieciach komputerowych istnieje podział na elementy składowe pasywne i aktywne.
Aby je odrurznić zadaj sobie pytanie czy dany element wpływa ( wytwarza, modyfikuje ) na sygnał czy nie. Jeżeli odpowiedź jest tak,
to jest to element aktywny.

Elementy aktywne Elementy pasywne
Rutery ( ang.: Routers ) Media transmisja ( kabelki )
Przełącznik ( ang.: Switches ) Panele krosownicze ( ang.: Patchpanel )
Punkty dostępowe ( ang.: Access Points ) Szafy dystrybucyjne ( ang.: Socket )
Serwery ( ang.: Servers ) Gniazda abonameckie ( ang.: Socket )
Fale radiowe (Media transmisyjne) Organizatory kabli
Korytka kablowe
